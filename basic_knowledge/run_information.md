# 比赛信息
你可以随时打开比赛信息界面来查看有关本赛局的信息。
比赛信息一共有4个界面，分别为牌型、盲注、优惠券、赌注。

## 牌型界面
![run_information1.png](/run_information1.png)
**牌型界面能看到你本赛局中你每种牌型的各类信息。**

①上方4个按钮用于切换4种界面。
②这里显示了每种牌型的等级，等级越高，对应牌型的基础筹码和基础倍率就越高。
③中间是牌型名称，将光标移动上去可以看到每种牌型的说明。
④这里是基础筹码和基础倍率，牌型等级越高基础筹码和基础倍率就越高，当你出牌时，会以对应牌型的基础筹码和基础倍率开始计分结算。
⑤这里是显示每个牌型在本赛局中你打出的次数，望远镜优惠券、超新星、方尖石塔、公牛会调用此值。
⑥返回打开比赛信息界面前的界面。
要注意的是，有3种牌型初始是隐藏的，它们对应的星球牌也是初始隐藏的，只有当你打出它们至少1次，才会出现在牌型界面中，它们的星球牌也才会出现在商店、卡包和女祭司中。

## 盲注界面
![run_information2.png](/run_information2.png)
你在盲注界面是无法对盲注进行任何操作的，只能进行信息的确认。
**当你击败boss盲注进入商店后，打开比赛信息的盲注界面，你就能看到下一底注的盲注信息。（1.0.1f以前的版本则不行，只能显示击败的boss盲注的底注信息）**

## 优惠券界面
![run_information3.png](/run_information3.png)
**你于本赛局中兑换的所有优惠券都会显示在这个界面中。**

## 赌注界面
![run_information4.png](/run_information4.png)
**显示你当前赛局已应用的赌注信息。**
