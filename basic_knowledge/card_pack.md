# Header
补充包一共有32种，其中有8种塔罗补充包，8种星球补充包，8种标准补充包，4种小丑补充包和4种幻灵补充包。
其中塔罗补充包、星球补充包、标准补充包每个等级对应的大小均不相同，从小到大依次分别为3选1，5选1和5选2，小丑补充包、幻灵补充包对应的则为2选1，4选1和4选2。每种补充包各自每个等级的数量比例为2：1：1，例如塔罗补充包有4种塔罗包，2种巨型塔罗包和2种超级塔罗包。
每场赛局**第1次**击败盲注后进入商店，一定售卖一个2选1的小丑包。
