# 选择盲注
当你按下盲注上方的选择按钮时，你就进入了盲注回合。
我们也可以按下盲注下方的跳过盲注按钮获得1个标签。
选择盲注上方的选择按钮后，先触发盲注的效果，如针、水，随后才触发小丑的效果，如窃贼、希科。
![selectblind.png](/selectblind.png)
①选择并挑战盲注，挑战成功后可以进入商店
②盲注所需的分数，达到这个分数就算挑战成功，达不到就算赛局失败
③跳过盲注可以获得1个对应的标签，大盲注和小盲注可以选择跳过，boss盲注则不能
④底注决定了盲注的分数，当通过底注8时，你就赢得了这场赛局，随后你可以选择挑战无尽模式或开启新赛局
⑤当你购买导演剪辑版、重构优惠券之后，重掷boss盲注的按钮会出现在这里
⑥你可以从比赛信息回顾你这场赛局的信息，牌型等级、即将面对的盲注、已购买的优惠券都可以在这里看到
⑦你随时可以点击你的牌组来查看你的完整牌组

# 盲注挑战回合
回合中，我们需要获得足够多的分数以击败盲注。
当然，我们在回合内能做的事情不止有获得分数能做，比如获得资金，改造牌组，小丑发育等等。

![blind_round1.png](/blind_round1.png)

①这里是小丑槽位，默认初始上限为5个，小丑牌存放于此处，**你可以在这里任意调整你的小丑的先后顺序**
②这里是消耗槽位，默认初始上限为2个，各种消耗牌存放于此处，需要指定目标或需要摧毁卡牌的消耗牌需要在回合内或者打开卡包时使用，不需要指定目标的消耗牌可以在其他界面使用
③这里是你的手牌，默认初始手牌上限为8，你可以同时选中最多5张手牌，**你可以在这里任意调整你的手牌先后顺序**
④这里能够看到你选中的牌的牌型、牌型等级、基础筹码和基础倍率，如果有翻面的牌被选中了，则会显示????（你尚未得知这是什么牌型）
⑤打出你所选的牌，并消耗1次出牌次数，进行计分，结算各类效果后抽牌至手牌上限
⑥舍弃你所选的牌，并消耗1次弃牌次数，不进行计分，结算各类效果后抽牌至手牌上限
⑦显示你剩余的出牌次数和弃牌次数，初始出牌次数为4，弃牌次数为3，当你出牌次数为0且得分还未达到盲注分数的要求，就算输掉这场赛局（骷髅先生在分数足够时可以避免此情况）
⑧手牌有两种排序方式，一种是点数优先，由大到小排列，同点数按黑桃红桃梅花方片的顺序摆放，一种是花色优先，按黑桃红桃梅花方片排列，同花色按点数由大到小摆放，默认状态下为前者，点击下方点数或花色按钮切换
⑨你可以将鼠标移至你的牌组上，上方会显示你牌组剩余的牌的数量，如下图所示：

![blind_round2.png](/blind_round2.png)

图中左边为花色，每种花色的牌都剩11张在牌库中，上方的为点数，比如卡组中还有4张A。如果你的卡组中有石头牌，则会显示在图中左上角黄框标识的区域。有时这个列表中的数字右边会跟上一个括号，括号中代表的是有效的卡牌数量，例如你将1张黑桃牌变为了万能牌，梅花这一栏就会显示11（12），这代表了你的卡组实际有11张梅花牌，但是有12张牌能够触发梅花相关的效果，梅花花色牌被boss盲注削弱了，此时梅花一栏就会显示11（0）。

# 计分结算
出牌计分时，先触发盲注效果（手臂、燧石等），再触发小丑的成长（绿色小丑、搭乘巴士等），然后结算游戏牌自身的效果（幸运牌、奖励牌、多彩等），接着结算牌面触发的小丑牌（奇数托德、微笑面孔、照片等），随后触发手牌效果（射月、致胜之拳、钢铁牌等），再结算直接提供倍率的小丑牌效果（抽象小丑、爆米花、拉面等），最后结算消耗槽位提供倍率（天文台）。
在以上的基础上，同一区域的卡牌按从左至右的顺序结算。

# 盲注挑战结束时
在盲注挑战结束时，有一个专门的时点用于触发各类小丑和游戏牌的效果，此时先结算小丑的效果，如鸡蛋与礼品卡的售价增加、爆米花与黑龟豆的效果减少、火箭的数值增长等，再结算游戏牌的效果，如蓝色蜡封、黄金增强，最后是易腐效果的倒数计数，如果计数为0就会立即削弱。
所以根据先后顺序，爆米花、黑龟豆即使有易腐效果，也会在最后1回合的削弱之前自毁。

# 盲注挑战结束后
结算击败盲注自带的奖励金，小丑槽位的各类经济丑（黄金小丑、9霄云外等）、投资标签与利息。
由于易腐削弱在前，所以黄金小丑一类在回合结束后提供经济的小丑在易腐的最后1回合已经无法提供作用了。
点击获得击败盲注的资金后，就会自动进入[商店](/basic_knowledge/shop)。
