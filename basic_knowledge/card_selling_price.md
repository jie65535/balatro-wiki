# Header
卡牌基础售价与商店价格有关，其计算公式如下：
（商店价格+版本价格）/2=基础售价
该售价向下取整，且最低为1￥。
版本会额外增加小丑的价格，分别为闪箔2￥，镭射3￥，多彩与负片均为5￥。

举例：
①一个商店价格为2￥的小丑，售出获得1￥，一个商店价格为3￥的飞溅，售出仍然是1￥。
②一个商店价格为6￥的占卜师，通过命运之轮获得闪箔后，其基础售价应为（6+2）/2=4￥。
③一个商店价格为2￥的小丑，购买了清仓特卖、清算优惠券后，售出仍然获得1￥。
④一个商店价格为6￥的占卜师，通过命运之轮获得闪箔，购买了清仓特卖优惠券后会影响占卜师的商店价格，其基础售价应为（6+2）/2×0.75=3￥，购买了清算优惠券后，其基础售价应为（6+2）/2×0.5=2￥
⑤不止小丑牌，星球牌、塔罗牌和幻灵牌都遵循这个计算方式，如售出商店价格为3的木星，售出会获得1￥，售出帕齐欧复制的负片塔罗牌，则会获得（3+5）/2=4￥。

**特例：鸡蛋、礼品卡提供的额外售价，是不受清仓特卖和清算优惠券的影响的，这两者提供的额外售价总是最后结算。**
