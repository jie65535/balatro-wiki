# 基础信息
·<center>![快乐安迪.png](/小丑牌/快乐安迪.png)</center>
> 名称：快乐安迪
稀有度：罕见
商店价格：<font color='EAC100'>$ 7</font>
能力：手牌上限-1，弃牌次数+3
{.is-info}

# 解锁条件
**在12或更少的回合内赢得一局**

# 特殊说明
**①由于<font color='EAC100'>绯红之心</font>会在每次出牌到抽牌之后才会更换削弱目标，当削弱<font color='EAC100'>快乐安迪</font>时出牌并抽牌后，<font color='EAC100'>绯红之心</font>切换削弱目标，手牌上限会变为当前手牌数-1，此时退出赛局并重进，手牌上限会变为与手牌数等同，获得了永久的手牌上限，这是一个bug**
