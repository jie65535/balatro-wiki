# 基础信息
·<center>![偶像.png](/小丑牌/偶像.png)</center>
> 名称：偶像
稀有度：罕见
商店价格：<font color='EAC100'>$ 6</font>
能力：每张打出的（对应花色）（对应点数）在计分时给予<font color=red>×2倍率</font>，每个回合结束时花色和点数都会改变
{.is-info}

# 解锁条件
**一次出牌至少获得1000000分数**

# 特殊说明
**①<font color='EAC100'>偶像</font>只会出现你完整牌组中有的牌，例如当你的完整牌组中只有<font color=Blue>梅花</font>A时，<font color='EAC100'>偶像</font>随机出现的牌只会是<font color=Blue>梅花</font>A**
**②当你的完整牌组中没有牌或者都是石头牌时，<font color='EAC100'>偶像</font>随机出现的牌将会为<font color=DarkViolet>黑桃</font>A**
