# 基础信息
·<center>![侵蚀.png](/小丑牌/侵蚀.png)</center>
> 名称：侵蚀
稀有度：罕见
商店价格：<font color='EAC100'>$ 6</font>
能力：当前的完整牌组数量每比初始的完整牌组数量少1张，这个小丑就获得<font color=red>+4倍率</font>（起始<font color=red>+0倍率</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>侵蚀</font>检测的是起始完整牌组，而废弃牌组起始完整牌组是40张而非52张，所以废弃牌组拿到侵蚀也并不会因比52张少立刻获得<font color=red>+48倍率</font></font>**
**②<font color='EAC100'>侵蚀</font>不会因为当前完整牌组数量比初始完整牌组数量多而反向减少倍率，即最少也是<font color=red>+0倍率</font>，不存在负的倍率**
