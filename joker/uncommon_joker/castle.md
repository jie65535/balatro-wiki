# 基础信息
·<center>![城堡.png](/小丑牌/城堡.png)</center>
> 名称：城堡
稀有度：罕见
商店价格：<font color='EAC100'>$ 6</font>
能力：每弃掉1张（对应花色）牌，这个小丑牌获得<font color='#0080FF'>+3筹码</font>，每个回合结束时花色都会改变（起始<font color='#0080FF'>+0筹码</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>只会复制<font color='EAC100'>城堡</font>的实际筹码，不会让<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>本身获得筹码变动**
**②<font color='EAC100'>城堡</font>只会出现你完整牌组中有的花色，例如当你的完整牌组中只有<font color=blue>梅花</font>时，城堡出现的花色只会是<font color=blue>梅花</font>**
**③当你的完整牌组中没有牌或者都是石头牌时，触发<font color='EAC100'>城堡</font>的花色将会为<font color=DarkViolet>黑桃</font>**
**④<font color='EAC100'>钩子</font>的随机舍弃2张手牌也可以触发<font color='EAC100'>城堡</font>的效果**
