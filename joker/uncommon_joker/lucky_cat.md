# 基础信息
·<center>![招财猫.png](/小丑牌/招财猫.png)</center>
> 名称：招财猫
稀有度：罕见
商店价格：<font color='EAC100'>$ 6</font>
能力：每次幸运牌被触发时，这张小丑获得<font color=red>×0.25倍率</font>（起始<font color=red>×1倍率</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>招财猫</font>只会在牌组中有幸运牌时才会出现**
**②<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>复制<font color='EAC100'>招财猫</font>只会复制其实际倍率，不会让<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>本身获得倍率变动**
**③在幸运牌的1次计分中触发中同时<font color=red>+20倍率</font>和<font color='EAC100'>+$ 20</font>只会获得1次倍率成长，但是在一张牌上的多次触发可以让<font color='EAC100'>招财猫</font>多次成长**

# 历史版本变更
**1.0.1f版本：每次幸运牌被触发时，这张小丑获得<mark><font color=red>×0.2倍率</font></mark>（起始<font color=red>×1倍率</font>）→每次幸运牌被触发时，这张小丑获得<mark><font color=red>×0.25倍率</font></mark>（起始<font color=red>×1倍率</font>）**
