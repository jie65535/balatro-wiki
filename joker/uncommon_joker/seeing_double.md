# 基础信息

> 名称：重影
稀有度：罕见
商店价格：<font color='EAC100'>$ 6</font>
能力：如果打出的牌中包含1张计分的<font color=Blue>梅花</font>花色牌和1张计分的其他花色牌，则给予<font color=red>×2倍率</font>
{.is-info}

# 解锁条件
**打出一手包含4张<font color=Blue>梅花</font>7的牌**

# 特殊说明
**①有万能牌、<font color='EAC100'>模糊小丑</font>的情况下，会以尽可能触发<font color='EAC100'>重影</font>的方式进行结算**
**②当打出的牌中所有的计分的<font color=Blue>梅花</font>花色牌都被削弱，就无法触发<font color='EAC100'>重影</font>**
