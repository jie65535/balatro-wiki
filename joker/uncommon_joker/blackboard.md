# 基础信息
·<center>![黑板.png](/小丑牌/黑板.png)</center>
> 名称：黑板
稀有度：罕见
商店价格：<font color='EAC100'>$ 6</font>
能力：如果留在手牌中的牌花色都是<font color=BlueViolet>黑桃</font>或<font color=Blue>梅花</font>，则给予<font color=red>×3倍率</font>
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①石头牌留在手中会使<font color='EAC100'>黑板</font>无法触发**
**②手中没有牌时可以触发<font color='EAC100'>黑板</font>**
**③留在手牌中的万能牌即使初始花色为<font color=Orange>方片</font>或<font color=Deeppink>红桃</font>，也仍然可以触发<font color='EAC100'>黑板</font>的效果**
