# 基础信息
·<center>![疯狂.png](/小丑牌/疯狂.png)</center>
> 名称：疯狂
稀有度：罕见
商店价格：<font color='EAC100'>$ 7</font>
能力：选择小盲注或大盲注时，这张小丑牌获得<font color=red>×0.5倍率</font>，并随机摧毁1张其他小丑牌（起始<font color=red>×1倍率</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>复制<font color='EAC100'>疯狂</font>只会复制其实际倍率，不会让<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>本身获得倍率变动**
**②<font color='EAC100'>疯狂</font>会优先摧毁不具有永恒的小丑牌，当你的其他小丑牌都是永恒的，<font color='EAC100'>疯狂</font>就不会摧毁小丑牌**
**③即使<font color='EAC100'>疯狂</font>没有成功摧毁小丑牌，也会正常获得倍率成长**
