# 基础信息
·<center>![绿色小丑.png](/小丑牌/绿色小丑.png)</center>
> 名称：绿色小丑
稀有度：普通
商店价格：<font color='EAC100'>$ 4</font>
能力：每次出牌这个小丑获得<font color=red>+1倍率</font>，每次弃牌这个小丑<font color=red>-1倍率</font>（起始<font color=red>+0倍率</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>复制<font color='EAC100'>绿色小丑</font>只会复制其实际倍率，不会让<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>本身获得倍率变动**
**②<font color='EAC100'>钩子</font>的弃牌也会使得<font color='EAC100'>绿色小丑</font>失去倍率**
**③<font color='EAC100'>绿色小丑</font>提供的倍率最低为<font color=red>+0倍率</font>，即使在<font color=red>+0倍率</font>情况下弃牌，<font color='EAC100'>绿色小丑</font>提供的倍率也不会变为负数倍率**

# 历史版本变更
**无**
