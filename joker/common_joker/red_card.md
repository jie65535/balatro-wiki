# 基础信息
·<center>![红牌.png](/小丑牌/红牌.png)</center>
> 名称：红牌
稀有度：普通
商店价格：<font color='EAC100'>$ 5</font>
能力：每次在补充包中按下跳过键，这张小丑牌获得<font color=red>+3倍率</font>（起始<font color=red>+0倍率</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>只会复制<font color='EAC100'>红牌</font>的实际倍率，不会让<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>本身获得倍率变动**
**②在超级补充包中，可以选择1张卡牌后再按下跳过键，仍然可以使这个小丑牌获得<font color=red>+3倍率</font>**


# 历史版本变更
**无**
