# 基础信息
·<center>![邮寄回扣.png](/小丑牌/邮件回扣.png)</center>
> 名称：邮寄回扣
稀有度：普通
商店价格：<font color='EAC100'>$ 4</font>
能力：每弃掉1张（对应点数）获得<font color='EAC100'>$ 5</font>，每个回合点数都会改变
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>邮寄回扣</font>只会出现你完整牌组中有的点数，例如当你的完整牌组中只有K时，<font color='EAC100'>邮寄回扣</font>出现的点数只会是K**
**②当你的完整牌组中没有牌或者都是石头牌时，<font color='EAC100'>邮寄回扣</font>的点数变更时将会固定为A**
**③<font color='EAC100'>钩子</font>的随机舍弃2张手牌也可以触发<font color='EAC100'>邮寄回扣</font>**

# 历史版本变更
**1.0.1f版本：每弃掉1张（对应点数）获得<mark><font color='EAC100'>$ 3</font></mark>，每个回合点数都会改变→每弃掉1张（对应点数）获得<mark><font color='EAC100'>$ 5</font></mark>，每个回合点数都会改变**
