# 基础信息
·<center>![未断选票.png](/小丑牌/未断选票.png)</center>
> 名称：未断选票
稀有度：普通
商店价格：<font color='EAC100'>$ 4</font>
能力：打出的第1张计分的牌额外触发2次
{.is-info}

# 解锁条件
**用高牌打赢boss盲注**

# 特殊说明
**①当你打出的第1张牌被削弱了，<font color='EAC100'>未断选票</font>就不会有任何效果，不会顺延至未被削弱的下一张牌**
