# 基础信息
·<center>![搭乘巴士.png](/小丑牌/搭乘巴士.png)</center>
> 名称：搭乘巴士
稀有度：普通
商店价格：<font color='EAC100'>$ 6</font>
能力：连续打出没有计分的人头牌时，这张小丑获得<font color=red>+1倍率</font>，失败将会重置为起始倍率（起始<font color=red>+0倍率</font>）
{.is-info}

# 解锁条件
**无需解锁**

# 特殊说明
**①<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>复制<font color='EAC100'>搭乘巴士</font>只会复制其实际倍率，不会让<font color='EAC100'>蓝图</font>和<font color='EAC100'>头脑风暴</font>本身获得倍率变动**
**②打出被削弱的人头牌时，也不会使<font color='EAC100'>搭乘巴士</font>重置为起始倍率**
**③当同时持有<font color='EAC100'>吸血鬼</font>和<font color='EAC100'>搭乘巴士</font>时，如果<font color='EAC100'>吸血鬼</font>的位置在<font color='EAC100'>搭乘巴士</font>左边，打出石头牌将其增强移除后，如果该牌是人头牌，则会<font color='EAC100'>搭乘巴士</font>会判定计分的有人头牌，于是会重置为起始倍率，如果将位置调换则不会重置**
