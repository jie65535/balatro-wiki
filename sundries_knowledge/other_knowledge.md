# 部分名词解释
人头牌：默认情况下人头牌指的是J、Q、K这3个点数的牌。当你有幻视时，所有牌都能触发人头牌相关的效果。
小丑牌削弱：削弱小丑牌将会使得小丑牌不能触发任何牌面上的效果，包括本身的效果和版本效果，但小丑牌本身的效果不会被清除，如罕见类型、售价，同时添加的贴纸效果也能正常生效，所以即使小丑牌被削弱，侠盗仍然能够通过削弱的小丑牌的售价提供倍率，罕见小丑仍然可以通过棒球卡提供乘倍，租赁贴纸仍然会在回合结束时扣除你<font color='EAC100'>$ 3</font>。
游戏牌削弱：游戏牌的削弱会使得无法触发任何牌面上的效果，无论是本身的增强、版本与蜡封，还是小丑牌相关的牌面触发，都无法触发，但是游戏牌本身的花色和点数仍然存在，还是可以参与牌型构成，即使构成5张牌型的牌全部被削弱，你仍然可以依靠牌型底分来搭配小丑牌获取分数。
不被允许：出现在灵媒、眼睛、嘴巴盲注当中，打出的手牌不符合boss盲注要求就会不被允许，此时此次出牌不会进行计分，但仍然能够给积分卡增加进度。
重复触发：指的是对满足条件的牌让其额外发动n次效果，比如牌面上的增强、版本和蜡封效果、小丑牌的牌面触发效果等，每次的发动如果有概率触发的事件（如<font color='EAC100'>8号球</font>、幸运牌），那么每次触发的概率都是独立的，互不影响。重复触发效果仅对计分时和回合结束时的效果有效。

# 收藏与解锁
在预设局（挑战局，输入种子的局）中无法解锁收藏。
当你于正常赛局中解锁了收藏，可以在同一赛局后续的进程中遇到。
挑战局无法解锁规则破坏者以外的成就。

# 初始牌组设定
**初始状态下，有4次出牌，3次弃牌，8手牌上限，5个小丑槽位，2个消耗槽位，初始资金<font color='EAC100'>$ 4</font>，牌组中有4种花色每种各有A~K这13种点数牌各1张。**
**每种牌组都有可能因自己的能力更改上述设定。**

# 手牌上限
**初始手牌上限为8，你可以通过小丑牌、消耗牌、优惠券等方式影响它的上限。**
手牌上限决定了你在回合内能够在同一时间持有的游戏牌数量，同时持有的游戏牌数量越多越容易达成大牌型。
同时要注意，手牌上限还会影响到秘术包和幻灵包，打开这两种补充包后会展示出你牌组中的部分游戏牌，你的手牌上限越多，能够展示的就越多，塔罗牌和幻灵牌能够选择的目标也就更多。
如果你的手牌上限减少后，手牌数量多于手牌上限，也不会弃掉手牌。

# 回合内的抽牌
当你在回合内增加手牌上限时（例如审判开出杂耍师、游吟诗人、黑龟豆，售出摔跤手削弱镣铐），如果你的手牌数小于手牌上限，会立刻尝试从牌组抽牌至手牌上限。
除了面对巨蟒，当你执行出牌或弃牌后，如果你的手牌数小于手牌上限，都会立刻从牌组抽牌至手牌上限。
使用火祭或倒吊人摧毁手牌后并不会立刻抽牌，但如果因此使得手牌张数变为0，会立刻尝试从牌组抽牌至手牌上限。

# 贴金标识
当你使用1张小丑通过底注8的boss时，该小丑会获得一个对应难度的赌注标识，代表你使用它通过了这一难度的赛局。
**必须是击败底注8的boss时持有的小丑，才会被贴上对应难度的赌注标识，在这之前将小丑卖掉了则无法贴上。在通过底注8后再购买小丑也不会贴上赌注标识了，即使你购买象形文字或岩画优惠券，重回底注8击败boss也一样无法贴上赌注标识。**
**在金注难度下，为了玩家更容易完成所有的成就，能够产出传奇小丑的灵魂在每个赛局第一次使用时会优先出现未拥有金注难度的赌注标识的传奇小丑。**

# 当前版本（1.0.1o）原版的特性与bug
**1）黄昏的效果描述错误，应为“最后一次出牌中打出并计分的牌额外触发1次”。**
**2）打开双倍标签给予的补充包，在回到主界面后继续赛局，会出现卡包中没有可使用消耗牌的bug，也无法跳过该补充包，成为死档，此时只能重开1把新赛局。**
**3）于底注8的boss盲注中，通关同时打出nan分数，游戏会崩溃。**
**4）当你打出nan分数后，最佳得分会变为nan，随后你只要打出任意分数都会刷新最佳得分。**
**5）当你打出nan分数后，在回到主界面后继续赛局，游戏会崩溃。**
**6）当你购买了幻象优惠券后，商店槽位内的游戏牌不会具有蜡封，但增强和版本可能会出现，这与描述不符。**
